## Wound State Quick Reference

| Condition | HP Threshold | Effect |
|---|---|---|
| **Not Wounded** | Above Seriously Wounded | No penalties |
| **Lightly Wounded** | Below max HP but above SW | No penalties |
| **Seriously Wounded** | At or below half HP | -2 to all Actions |
| **Mortally Wounded** | 0 HP or below | Unconscious; Death Save each Turn |
| **Dead** | Failed Death Save | Character is dead |
